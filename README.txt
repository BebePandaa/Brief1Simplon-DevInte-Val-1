**Description**
Mon objectif était de reproduire mon CV en utilisant du code UNIQUEMENT en format HTML.
Il existe une version desktop (ordinateur) et une version mobile.
Cette page recense mes informations personnelles (mon profil, mes passions, ma personnalité), mes différentes expériences professionnelles et mes qualifications / diplômes.

On peut aussi y retrouver un formulaire d'inscription contenant : Nom, Prénom, Age, Sexe, Adresse mail, Adresse postale, Ville, Métier, Nationalité, Date de naissance et Pays de naissance.
Le tout fait en HTML uniquement !
